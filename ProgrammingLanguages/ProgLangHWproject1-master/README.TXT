USER INSTRUCTIONS:
This Program takes information from a text file and calculates a predicted winner of a state's primary election. The format of the file must be as follows: 
Line 1: Two letter abbreviation of the current state
Line 2+: Two letter representation of candidate, space, Two letter abbreviation of the candidates home state, space, integer value of how many commercials were run in the current state for the candidate, space, integer value of how many days the candidate spent campaigning in the state, space, integer value of how many primary electuibs the candidate has won this year, next line
